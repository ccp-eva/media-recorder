Silent Webcam Recorder

1 Utility Functions

The following functions are loaded into the window object, and thus work from everywhere.
  • toggleModal() ⮕ show/hide the modal UI
  • startWebcamStream() ⮕ starts a webcam stream w/o recording it
  • stopWebcamStream() ⮕ stops all active webcam streams
  • startWebcamRecorder() ⮕ starts a webcamstream (if not active) and starts recording
  • stopWebcamRecorder() ⮕ stops the recording, creates the videofile, stops the webcam stream
  • playbackRecording() ⮕ opens the modal to view the recording
  • uploadVideo(filename) ⮕ uploads the recording (filename defaults to ISO 8601 timestamp)

2 Demonstration

You can simply tack the above function to any UI element. The samples belows use buttons.
You can run the buttons sequentially to see the effect.
  1. ⮕ No webcam stream loaded (empty modal)
  2. ⮕ starts a webcam stream w/o recording it
  3. ⮕ You should see yourself
  4. ⮕ stops all active webcam streams
  5. ⮕ starts a webcamstream (if not active) and starts recording
  6. ⮕ You should see yourself (you get recorded now)
  7. ⮕ stops the recording, creates the videofile, stops the webcam stream
  8. ⮕ opens the modal to view the recording
  9. ⮕ uploads the recording (filename defaults to ISO 8601 timestamp)
